# dq1 — ドラクエ1風テキストRPG

魔王ゾーマが世界を闇に包んでいる。勇者よ、立ち上がれ！

## 起動方法

```bash
python lab.py gris dq1
```

または直接：

```bash
cd commands/gris/dq1
node index.js
```

## ゲーム概要

- 勇者の名前を入力して冒険開始
- 4つのエリアを移動しながら敵と戦い、レベルアップ
- ラスボス「ゾーマ」を倒すとエンディング

## マップ

```
[魔王城]  ← ゾーマがいる
   │
[洞窟]    ← 強い敵が出る
   │
[大草原]  ← 草原の敵
   │
[城下町]  ← みせ・やどがある（セーフゾーン）
```

## コマンド（フィールド）

| コマンド | 効果             |
|----------|------------------|
| `n`      | 北へ移動         |
| `s`      | 南へ移動         |
| `s`      | みせ（城下町）   |
| `i`      | やど（城下町）   |
| `st`     | ステータス表示   |
| `save`   | セーブ           |
| `load`   | ロード           |
| `map`    | マップ表示       |
| `q`      | ゲーム終了       |

## コマンド（戦闘）

| コマンド | 効果                             |
|----------|----------------------------------|
| `1`      | たたかう（物理攻撃）             |
| `2`      | じゅもん（ホイミ or ギラ）       |
| `3`      | どうぐ（やくそうを使う）         |
| `4`      | にげる（75%成功）                |

### じゅもん

| 呪文   | MP消費 | 効果             |
|--------|--------|------------------|
| ホイミ | 3      | HP を 28〜38 回復 |
| ギラ   | 5      | 敵に 22〜32 ダメージ |

## 登場モンスター

| 名前       | エリア | HP  | 特徴           |
|------------|--------|-----|----------------|
| スライム   | 草原   | 12  | 最弱。経験値少 |
| ドラキー   | 草原   | 22  | 素早い         |
| キメラ     | 洞窟   | 42  | 中程度の強さ   |
| ゴーレム   | 洞窟   | 65  | 硬くて強い     |
| ゾーマ     | 魔王城 | 200 | ラスボス       |

## 成長システム

- Lv.1〜10 まで成長
- 敵を倒すと経験値とゴールドを獲得
- レベルアップ時：HP・MP・攻撃力・防御力が上昇
- ゾーマ攻略推奨レベル：Lv.8〜10

## セーブデータ

`commands/gris/dq1/save.json` に保存されます。  
タイトル画面の `[l]つづきから` か、フィールドで `load` コマンドで読み込めます。

## 動作環境

- Node.js 18 以上
- 外部依存なし（標準ライブラリのみ）
- ANSI カラー対応ターミナル推奨

## 演出

- タイプライター風テキスト表示
- ANSI カラー（黄・白・赤・緑・青）
- 枠付きメッセージ（戦闘開始・レベルアップ・ゲームオーバー・エンディング）
