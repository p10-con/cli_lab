# pendulum

二重振り子のカオス的な軌跡をターミナルにリアルタイム描画する。
初期角度がわずかに違うだけで全く異なる軌跡を描く（カオス理論の典型例）。

## 使い方

```bash
python lab.py gris pendulum
```

Ctrl+C で終了。

## 仕組み

- 二重振り子の運動方程式（m1=m2=1, L1=L2=1, g=9.81）をオイラー法で数値積分
- 1フレームにつき15ステップのサブステップで精度を確保
- 第2ボブの軌跡を緑のグラデーションで残す
- ANSI エスケープコードで文字を絶対座標に配置して描画

## 依存

なし（Node.js 標準ライブラリのみ）
