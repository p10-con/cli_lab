# aurora

ターミナルに降り注ぐオーロラアニメーションです。  
通常はモノクロの ASCII アートで揺らめき、カーソルを動かすと周辺だけ aurora カラーが灯ります。

## 実行方法

```bash
python lab.py gris aurora
```

## 操作方法

| 操作 | 効果 |
|------|------|
| 矢印キー ↑ ↓ ← → | カーソルポインタを移動 |
| マウスを動かす | ポインタが追従（対応ターミナルのみ） |
| **Ctrl+C** | 終了 |

## カラーエフェクト

- **カーソル中心**: 鮮やかなオーロラカラー（緑→シアン→青→紫→ピンク）が列位置と時刻に合わせてゆっくり変化
- **中心から外側**: 鮮やかさが白→グレーへと自然にフェードアウト
- **カーソルから遠い領域**: モノクロの ASCII オーロラ

## 動作環境

- Node.js 18 以上
- True Color（24-bit カラー）対応ターミナル推奨
  - macOS: Terminal.app / iTerm2
  - Windows: Windows Terminal
  - Linux: 各種 VTE ターミナル
- ターミナルを広げるほど（推奨: 120列 × 36行 以上）美しく見えます

## 依存関係

外部パッケージなし。Node.js 標準ライブラリのみ使用しています。

## 仕組み

複数の正弦波を重ね合わせることでオーロラカーテンの揺らぎを表現しています。  
各セルの輝度をガウス減衰関数で計算し、ASCII 文字（` .·:╎│┃║█`）に変換して描画します。  
カーソル周辺では HSL カラーモデルで色相を列位置と時刻から算出し、  
proximity（近さ）の 1.8 乗でカラーと白のブレンド比を制御しています。
