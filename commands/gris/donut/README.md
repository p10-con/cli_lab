# donut

3Dトーラス（ドーナツ）をASCII文字でレンダリングして回転させる。
Andy Sloane の有名な "donut.c" を Node.js で実装し、カラー化したもの。

## 使い方

```bash
python lab.py gris donut
```

ターミナルを広くすると見栄えが良くなる。Ctrl+C で終了。

## 仕組み

1. トーラス表面を θ（断面）× φ（旋回）でパラメトリックサンプリング
2. X軸・Z軸の回転行列を適用して3D座標を得る
3. 透視投影でターミナル座標（xp, yp）に変換
4. 法線ベクトルと光源方向の内積で輝度 L を計算
5. L の値で ` .,-~:;=!*#$@` から文字を選んで描画
6. zバッファで手前の面が奥の面を隠す（奥行き判定）

## 依存

なし（Node.js 標準ライブラリのみ）
